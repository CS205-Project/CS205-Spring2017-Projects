Overview

Below are performance plots of evacuation of 500-15000 people. These plots show time to calculate the evacuation, the speedup gained, and the efficiency. All of these simulations were run with 1,2,4,8,16,32,64 cores. From these plots we can see that the efficiency decreases as we increase the number of processors. However, we get close to linear speedup for 16 or less cores in most cases.
